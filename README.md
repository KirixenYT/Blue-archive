## 部署方式

> **推荐环境：**
>
> node > 16.16.0  
> npm > 8.15.0

1. 安装yarn

```bash
# 安装 yarn
npm install -g yarn
```

2. 克隆此项目到本地
3. 在项目根目录下运行

```bash
# 安装依赖
yarn install

# 预览（开发环境）
yarn dev

# 构建
yarn build

# 预览（生产环境）
yarn preview
```

> 构建完成后，静态资源会在 **`dist` 目录** 中生成，你可以将 **`dist` 目录中的文件**上传至服务器，也可以使用 `Vercel`、`Netlify` 等静态托管平台一键导入并自动部署
